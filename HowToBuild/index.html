How to Build Zephyr

Requirements

  • Git
  • Gradle
  • Java 7
  • CDH 5.0.0 (MRv1) - Untested on other Hadoop distributions (but it probably works!)
  • Download Zephyr and Zephyr Sample Project

How to Build Zephyr Project

    
        $ cd zephyr
        $ gradle clean install
        $ cp -r build/libs/ /opt/libs/

How to Build Zephyr Sample Project

    
        $ cd zephyr-sample-project
        $ gradle clean dist -PcdhVersion=cdh5
Note: The flag -PcdhVersion can be set to either cdh4 or cdh5. To specify a specific version, edit the file zephyr-sample-project/sample-project-mapreduce/gradle.properties.
