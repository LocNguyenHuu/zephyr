How to Build Zephyr

Requirements

  • Git
  • Gradle
  • Java 7
  • Cloudera CDH 5.0.0 (MRv1) OR CDH 4.5.x (MRv1) - Untested on other Hadoop distributions (but it probably works!)
  • Download Zephyr and Zephyr Sample Project 
    
        $ cd zephyr
        $ gradle clean install
        $ cd ../zephyr-sample-project
    

Then, depending on your version of CDH;
    
        $ gradle clean dist -PcdhVersion=cdh5
    
or
    
        $ gradle clean dist -PcdhVersion=cdh4
Note: To specify a different reversion of CDH, edit the file zephyr-sample-project/sample-project-mapreduce/gradle.properties.
